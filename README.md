# Nightlight
A small game made for Jame Gam 2024.

# Features
- Pixel Art Style
- Combat system featuring... a candle?
- Dreamy Atmosphere
- Puzzles
- Nightmares (Enemies)
- Quirky Game Mechanics
- Expansive Map
