# Nightlight
A small game originally made for Jame Gam 2024, now an ongoing project.
THIS IS UNFINISHED AND IS CURRENTLY IN THE BETA STAGES OF DEVELOPMENT.

# Features
- Pixel Art Style
- Combat system featuring... a candle?
- Dreamy Atmosphere
- Puzzles
- Nightmares (Enemies)
- Quirky Game Mechanics
- Expansive Map
